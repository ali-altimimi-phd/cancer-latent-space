# Cancer Latent Space Analysis

This repository represents an **exploratory phase** of the broader *Cancer Complexity Analysis* project, focusing on latent space modeling of gene expression data using machine learning methods.

---

## Relationship to Primary Project

This work builds directly on the preprocessing and data integration pipeline implemented in:

→ https://github.com/ali-altimimi-phd/global-cancer-complexity

Specifically, this repository was designed to operate on datasets such as:

* `global_cancer_eset_list.RData`

which are produced by the preprocessing pipeline in the primary repository.

---

## Purpose (Exploratory Phase)

The objective of this repository was to investigate whether classical gene expression–based analyses of cancer could be reinterpreted through modern representation learning approaches, including:

* Latent space modeling using **variational autoencoders (VAE)**
* Geometric analysis of cancer classes in learned representations
* Comparison between:

  * biological complexity (entropy / structural measures)
  * latent geometric structure

This work complements the original complexity/entropy framework by introducing a **representation-learning perspective**.
