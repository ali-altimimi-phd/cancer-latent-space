# Cancer Latent Space Analysis

This repository contains an in-progress extension of the *Cancer Complexity Analysis* project, focusing on latent space modeling of gene expression data.

## Relationship to Primary Project

This work builds directly on the preprocessing and data integration pipeline implemented in:

https://github.com/ali-altimimi-phd/global-cancer-complexity

Specifically, this repository uses the dataset:

- `global_cancer_eset_list.RData`

which is produced by the preprocessing pipeline in the primary repository.

## Purpose

The goal of this extension is to reinterpret classical gene expression–based analyses of cancer through modern machine learning approaches, specifically:

- latent space modeling using variational autoencoders (VAE)
- geometric analysis of cancer classes in learned representations
- comparison between biological complexity (entropy/structure) and latent geometric structure

This work complements the original complexity/entropy framework by introducing a representation-learning perspective.

## Current Status

This repository is under active development.

- Data preparation and model training pipelines are being established
- Latent representations are being analyzed for within- and between-cancer structure
- Integration with the classical complexity framework is ongoing

## Notes

- This repository depends on outputs generated by the primary project
- It is not intended to function as a standalone pipeline at this stage
- Reporting and interpretation will be integrated with the broader project over time
